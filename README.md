# 📰 PulseNews — Современный Новостной Портал

Инновационный новостной веб-сервис, построенный на стеке **Nuxt 3 (Vue 3)**, **Pinia** и **Tailwind CSS**. Проект включает адаптивный публичный портал и полноценную SPA-административную панель для контент-менеджеров.

## ✨ Реализованный Креативный Функционал
1. **Parallax Effect:** На заглавном экране публичной страницы с использованием адаптивной CSS-сшивки.
2. **Skeleton Loaders:** Плавное состояние ожидания карточек новостей при асинхронных запросах к API.
3. **Система Уведомлений (Toasts):** Моментальный визуальный фидбек при совершении CRUD операций в панели управления.
4. **Dark/Light Mode:** Динамическое переключение визуального стиля (Tailwind `dark:`).
